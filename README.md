# Online_book_store_sql
A PostgreSQL-based Online Bookstore database project demonstrating table design, relationships, and SQL queries for real-world data analysis.

This project is a relational database system designed for an Online Bookstore, implemented using PostgreSQL. It focuses on database design, data relationships, and SQL queries to analyze business data such as book inventory, customer activity, and sales performance.

ðŸ”¹ Database Schema

The project includes three main tables:
Books: Stores book details such as title, author, genre, price, publication year, and stock.
Customers: Contains customer information including name, contact details, and location.
Orders: Records customer purchases with references to books and customers using foreign keys.

Aggregate functions: SUM(), AVG(), COUNT()
JOIN operations (INNER JOIN, LEFT JOIN)
Grouped analysis using GROUP BY and HAVING
Real-world business queries such as revenue calculation, stock tracking, and customer spending analysis
